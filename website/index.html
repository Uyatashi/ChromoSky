How it works

  • "Shoot" the sky

    Take a photo of the sky using your camera. Then submit the data to our desktop application.

  • Analyze the data

    The desktop application analyzes the data and publishes its results on this website.

  • View results

    View a Google map with all the information for each location from which the pictures were processed and analyzed.
